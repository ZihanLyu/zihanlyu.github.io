Creative Manual:

I created a pomodoro timer 🍅 using starter code found online.
The 'Creative' tab will navigate you to the 'Login' page.
Login
First-time users have to create an account.
Returning users have to login with the correct credentials.
The pomodoroTimer allows user to choose from a 25 minutes timer,
a 5 minutes short-break, and a 10 minutes long-break.
Statistics
The 'Statistics' tab will navigate the user to their statistics page.
There will be 'Total pomodoro' and 'Total minutes' recorded.
Also, there is a 'Weekly Activity' line plot, and a 'Timer Distribution' pie plot.
Logout
The 'Logout' tab allow user to logout from their account and goes back to the login page.
